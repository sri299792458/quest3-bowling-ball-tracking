# Quest Bowling Final Demo Video

Remotion project for the final narrated project demo. It combines the recorded Quest/live-pipeline clips, still frames, generated narration, and section labels into the final 1080p video.

The rendered video is hosted here:

<https://drive.google.com/file/d/1ChvX2wGKDNvMor3lKa0ZUYFAq08_IGWq/view?usp=sharing>

## Setup

```powershell
npm install
```

## Preview

```powershell
npm run dev
```

## Render

```powershell
npx remotion render QuestBowlingDemo out/quest_bowling_demo_final.mp4
```

Generated files under `out/` and installed dependencies under `node_modules/` are ignored by Git.
